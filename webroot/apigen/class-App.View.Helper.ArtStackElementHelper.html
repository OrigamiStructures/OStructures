Class ArtStackElementHelper

ArtStackElement encapsulates rendering rules to select output elements for the view

To insure that every time an artwork is shown on the screen, the rendering flows down through a single simple path that deposits the DOM structural nodes and loops in the appropriate places.

This same set of structure-control elements is used for Create, Refine, and Review display. So there are many points in the output where rules must be evaluated to decide which final output should be used. Since there a ton of small elements involved, all the rules were moved into this helper.

These are very general rules that decide whether to use display or fieldset elements. Those elements, in many cases, need to make much more detailed output choices based on edition type. Those more compelx rules are found in the EditionFactoryHelper heirarchy.
